╔══════════════════════════════════════════════════════════════════════════════╗
║                        🔄 FLUXO DE TRABALHO                                  ║
╚══════════════════════════════════════════════════════════════════════════════╝

1️⃣  Cliente faz pedido e paga via PIX
    └─> Status: "pending" → "completed"

2️⃣  Pedido aparece no painel admin com status "PAGO"
    └─> Botão verde "Enviar" fica disponível

3️⃣  Admin clica em "Enviar"
    └─> Sistema envia para SMMMIDIA automaticamente
    └─> Status muda para "processing"

4️⃣  SMMMIDIA processa e entrega
    └─> Status muda para "delivered"

PASSO 1: Criar usuário admin
   cd server
   npm run generate-admin

PASSO 2: Executar SQL no Supabase
   └─> Copie o SQL gerado
   └─> Execute no SQL Editor do Supabase

PASSO 3: Fazer login
   └─> Acesse: http://localhost:5173
   └─> Faça login com email e senha do admin
   └─> Será redirecionado automaticamente para /admin
